# Design Guidelines - Taste-Skill

## Before Creating Any Design

Before creating, styling, or designing any UI components or pages, **always consult the taste-skill first** using:

```
/design-taste-frontend
```

This ensures all designs follow modern, premium design principles with proper:
- Animations and transitions
- Spacing and layout
- Visual hierarchy and quality
- Premium aesthetic standards

### When to Consult Taste-Skill

- Creating new components
- Designing pages or layouts
- Adding styling or visual improvements
- Modifying UI elements
- Building premium visual interfaces

### Available Taste-Skill Commands

The following design skills are installed:
- `/design-taste-frontend` - Core premium design guidance (main skill)
- `/soft-skill` - Premium aesthetic with spring animations
- `/minimalist-ui` - Clean, editorial interfaces
- `/industrial-brutalist-ui` - Mechanical, Swiss-inspired design
- `/high-end-visual-design` - Luxury visual design
- `/redesign-existing-projects` - Audit and improve existing designs
- `/full-output-enforcement` - Complete, production-ready outputs
- `/stitch-design-taste` - Google Stitch-compatible design rules

### Workflow

1. Describe what you want to design
2. Run `/design-taste-frontend` to get design guidance
3. Follow the guidance while implementing
4. Create beautiful, premium interfaces
